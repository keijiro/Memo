# Bug Reporting for Unity

次のようなプロンプトを使って ChatGPT で清書を行う。

```
Unity におけるバグと思わしき挙動をバグレポートとして提出したいと思います。この内容のプルーフリードをお願いします。かなり雑に英文を書いたため、分かりにくい部分や、英文として不自然な部分など、問題が多々あると思います。そういった部分は大幅に書き直して構いません。

なお、Unity のバグレポートは、Summary と “1. What happened” と “2. How can we reproduce it using the example you attached” の３要素から構成されるというルールがあります。この構成は崩さないようにしてください。また、Summary は "(Summary)" とプレースホルダのみ置いてありますので、ここに「１行まとめ文」を作って埋めてください。

---

(Summary)

1. What happened

...

2. How can we reproduce it using the example you attached

...
```
